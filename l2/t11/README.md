## Структура

`main.rs` - инициализация и запуск сервера

`contloller.rs` - хэндлеры запросов

`model.rs` - функции работы с моделью данных

## Запуск

Для запуска :

```bash
cargo run
```

## Хэндлеры

Пример тела запроса для /create_event и /update_event 
```
{
    "event_id": 1,
    "name": "event3",
    "date": "2023-01-01",
    "user_id": 1
}
```
### Хэндлеры:

- добавление ивента в память
```
POST /create_event
```
- обновить данные об ивенте в памяти
```
POST /update_event
```
- удалить данные об ивенте в памяти
```
POST /delete_event
```
- получить данные об ивенте в дне по дате из памяти
```
GET /events_for_day
```
- получить данные об ивенте в неделе (к которой принадлежит день в дате) по дате из памяти
```
GET /events_for_week
```
- получить данные об ивенте в месяце по дате из памяти
```
 GET /events_for_month
```
